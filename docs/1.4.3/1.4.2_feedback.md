# 1.4.3 업데이트 요구사항

1.4.3 업데이트를 진행하기 전, 1.4.2 버전에서 사용자 피드백과 운영자의 피드백을 작성한 문서입니다. 1.4.3 업데이트 계획을 세우기 전, 사용자 피드백과 운영자 피드백을 참고하여 'docs/1.4.3/plan.md' 파일로 업데이트 계획을 작성해주세요.

## 사용자 피드백

### 개선 요청 피드백
    - PC 환경 사용자1 : 24,27인치 등 디스플레이의 크기를 고려하지 않은 UI 디자인이라고 느껴집니다. 
    - PC 환경 사용자2 : 타이머가 다 지나가도 RPS 상태가 변경되지 않습니다. 
    - PC 환경 사용자3 : 마우스 위치를 변경해도 아바타가 따라가는 딜레이가 있는 것 같습니다.
    - PC 환경 사용자4 : 하드웨어 스펙 , 네트워크 상태 모든 게 더 좋은 상태인데, 다른 컴퓨터에 비해서 렉이 걸립니다.
    - 모바일 환경 사용자 : PC 환경에 비해서 너무나도 아바타 화질이 떨어집니다.
    - 공통 사항1 : RPS 상태의 가시성이 떨어지는 것 같습니다.
    - 공통 사항2 : 방 만들기로 친구들끼리 게임을 하다가 죽으면 다시하기를 누른 경우 다른 방으로 이동됩니다. 친구들이랑 같은 방에 계속 하고 싶습니다.
    - 공통 사항3 : 게임을 시작한 순간에 렉이 좀 심하고, 1초 정도 지나면 완전히 부드러워집니다.

### 버그 피드백
    - 방만들기 -> 코드 생성 -> 링크 복사 -> 접속을 한 경우 링크가 '주소?=CODE=코드' 형식으로 생성이 됩니다. 이때 로비로 이동 후 다시 방을 만든 경우 사용자에게는 코드가 변경된 채로 전송이 되지만, 현재 제 주소는 변경되지 않습니다. 하지만 제 주소가 이전 코드더라도 만나지는데에는 문제가 없습니다.

### 긍정 피드백
    - 게임이 매우 재밌습니다. 
    - 엄청나게 부드럽게 작동합니다. 주변에서 렉이 심하다고 해서 걱정했는데 저는 전혀 그렇지 않습니다.
    - 긴장감이 넘칩니다.
    - 모바일,PC 환경 모두에서 만날 수 있다는 것이 정말 좋습니다.

## 운영자 피드백
    - 삼성 브라우저는 저성능 브라우저가 아니지만, 저성능 브라우저로 판단하고 크롬 브라우저로 이동시키는 정책이 존재하고 있습니다. 이전에 삭제를 요청했지만, 반영되지 않은 것 같습니다. 브라우저 이동 정책은 오직 인앱브라우저인 경우 -> 외부 브라우저로 이동하도록 권장하는 것 뿐입니다.
    - 다양한 국가의 사용자들이 조금 더 편하게 문의를 할 수 있는 채널이 생기면 좋을 것 같습니다.
    - 모든 사용자들이 피드백을 남기는 것이 아니기 때문에, 사용자가 피드백을 남기지 않더라도 사용자가 얼마나 부드럽게 게임이 되고 있는지 게임에는 문제가 없는지에 대한 정보를 얻고 싶습니다. 방법이 있는지 찾아주세요.
    - 일부 컴퓨터에서는 아무 문제 없이 진행되지만, 일부 컴퓨터에서는 문제가 발생되고 있습니다. 세계 최고의 개발자로서 개선 방법을 고려해주세요.
