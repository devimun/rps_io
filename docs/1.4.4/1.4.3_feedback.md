# 1.4.3 버전 리포트

## 개요
1.4.3 버전에서 주된 수정 목표였던 초기 렉 개선이 이루어지지 않아 1.4.4 버전 업데이트 목표를 작성합니다. [크롬 F12 퍼포먼스 테스트 녹화](./Trace-20260105T002129.json) 를 해석하고 현재 프로젝트에서 문제를 분석하고 개선 계획을 제시합니다.

## 필독사항
1. 절대 수정으로 인해서 게임성에 저하가 발생해선 안됩니다.
2. slihter.io와 같은 최고의 io 게임이 될 수 있는 방법을 제시합니다.
3. 현재 구조가 잘못된 점이 있다면, 큰 규모의 작업일지라도 진행하는 것을 목표로 합니다.
4. 절대 임의로 작업을 진행하여선 안됩니다.
5. 퍼포먼스 트레이스 파일의 규모는 매우 크므로 프레임 드랍이 발생하는 지점만을 분석하는 것을 목표로 합니다. 만약 해석에 어려움이 있다면, 프레임 드랍이 발생하는 지점을 도출하는 프로그램을 하나 더 제작하여 사용합니다.
