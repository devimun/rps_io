# 📄 Project Specification: ChaosRPS.io 
## 1. 프로젝트 개요
* **슬로건:** "내 맘대로 안 되는 대환장 가위바위보 서바이벌"
* **장르:** 실시간 멀티플레이어 IO 액션
* **플랫폼:** 모바일 웹 / PC 웹 
* **핵심 재미:** 랜덤하게 변하는 상성(가위/바위/보)을 이용한 쫓고 쫓기는 심리전과 피지컬 싸움.
* **타겟 유저:** 짧은 시간(킬링타임)에 친구들과 가볍게 내기를 하거나 즐기고 싶은 1020 세대.

## 3. 핵심 기능 명세 (Core Features)

### A. 게임 룰 (Game Loop)
1.  **랜덤 변신:** 모든 플레이어는 N초(예: 1.5초)마다 가위/바위/보 중 하나로 강제 변신. (변신 0.5초 전 시각적 예고 필수)
2.  **상성 판정:**
    * **승리:** 가위>보, 보>바위, 바위>가위 → 상대방 흡수(점수 UP, 크기 UP).
    * **무승부:** 같은 상성 → 튕겨나감 (넉백 효과).
    * **패배:** 상성 열위 → 사망 (Game Over).
3.  **이동 조작:**
    * **PC:** 마우스 커서 추적.
    * **Mobile:** 화면 터치 드래그 (가상 조이스틱).
4. **랭크표시**
    * 우측 상단에 랭크 표시(모바일/PC 반응형 UI 필수)
5. **사용 방법 안내**
    * 사용 방법 안내, 사용자가 끄거나 다시 보지 않기를 할 수 있음.

### B. 로비 & 매칭 시스템 (Entry Flow)
* **3가지 입장 방식:**
    1.  **[⚡ 바로 시작]:** 봇과 사람이 섞인 공방(Public Room)으로 즉시 난입. (매칭 대기 시간 0초)
    2.  **[🏠 방 만들기]:**
        * **옵션:** `[v] AI 봇 채우기` 체크박스 제공.
            * *체크 시:* 친구들과 함께 봇을 잡는 '난전 모드'.
            * *체크 해제:* 오직 친구들끼리만 싸우는 '진검승부 모드'.
        * **액션:** 방 생성 후 즉시 입장 + 초대 코드 생성.
    3.  **[🔑 코드 입장]:** 6자리 초대 코드 입력 시 해당 비공개 방으로 입장.
* **URL 다이렉트 입장:** `chaosrps.io/?code=A1B2` 링크 접속 시, 로비 생략하고 닉네임만 입력 후 해당 방으로 즉시 투입.

### C. 봇(AI) 시스템
* 방 정원(20명) 유지를 위해 유저가 퇴장하면 봇 투입.
* 유저가 입장하면 봇 1명 즉시 퇴장 (스위칭 시스템).

### D. 미니맵 시스템
* 다른 사용자의 위치를 보기 위해, 내 위치를 기준으로 인근 맵을 보여주는 미니맵이 존재해야 합니다.


### E. 주의사항
* 20명이 사용하기에 적합한 월드맵을 사용해야 합니다.
* 사용자가 마우스(모바일의 경우 대시 버튼)를 꾹 누르면 빨라질 수 있어야 합니다. 단, 달릴 수 있는 시간은 1초 이내입니다.
* 사용자들이 태어나자마자 죽을 일이 없기 위해서 사용자가 게임에 들어가는 경우, 사람들로부터 먼 곳에서 태어나야 합니다.

## 4. UI/UX 및 환경 대응 (Cross-Platform Strategy)

### A. 인앱 브라우저 성능 이슈 대응 (Critical)
카카오톡, 인스타그램 등 인앱 브라우저는 성능이 낮고 화면 비율이 좁아 게임 플레이에 제약이 있음.

1.  **성능 경고 모달 (Performance Alert):**
    * **Trigger:** User Agent 감지 결과 `KAKAOTALK`, `NAVER`, `LINE`, `Instagram` 등이 포함된 경우.
    * **UI:** 화면 상단에 닫기 가능한 경고 배너/모달 노출.
    * **Copy:** *"⚠️ 현재 브라우저는 게임이 끊길 수 있습니다! 더 쾌적한 플레이를 위해 외부 브라우저(크롬, 사파리)로 열어주세요."*
    * **Action:** `[외부 브라우저로 열기]` 버튼 제공 (Android Intent / iOS 클립보드 복사 유도).
2.  **그래픽 자동 하향 (Low Spec Mode):**
    * 인앱 브라우저 감지 시, 파티클, 그림자 등 고사양 옵션을 강제로 끄고 프레임을 확보.

### B. 모바일/PC 반응형 컨트롤
* 접속 기기를 감지하여 UI 레이아웃 자동 변경.
* **Mobile:** 전체 화면(Full Screen) 유도 버튼 제공, 터치 영역 최적화.

## 5. 바이럴 장치 (Viral Mechanics)
